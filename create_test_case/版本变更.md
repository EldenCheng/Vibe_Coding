# 版本变更记录

## v1.1 (2026-06-17)

### Bug修复

- **修复sheet名称清理导致的Summary公式错误**：当mode_name包含特殊字符（如`/`）时，Summary sheet中的COUNTIF公式会引用未清理的sheet名称，导致`#VALUE!`错误
  - 新增`_sanitize_sheet_name()`方法，统一处理sheet名称清理逻辑
  - Summary sheet现在使用清理后的名称作为显示值和公式引用
  - 确保Summary中的sheet名称与实际数据表名称完全一致

## v1.0 (2026-06-16)

### 新增功能

- **文件类型检测**：自动检测输入文件类型（PDF/图片），分发到对应处理函数
- **PDF转图片**：使用PyMuPDF将PDF每页转换为JPG图片，保存到output/images/目录
- **图片处理**：支持直接输入图片文件（jpg/png/bmp/gif），原样复制到output/images/目录
- **API调用封装**：封装大模型API调用，支持图片模式（多模态）和文本模式两种调用方式
- **配置管理**：从config/auth.json读取API认证信息（base_url, api_key）
- **提示词管理**：
  - 基础分析提示词始终作为第一部分
  - 可选提示词通过config/prompts_mapping.json配置，支持单选或多选
  - 临时提示词通过命令行参数输入，作为最后一部分
  - 动态拼接生成最终提示词
- **分析处理**：逐个读取图片，调API生成分析结果（markdown格式），保存到output/analysis/目录
- **测试用例生成**：从分析结果（markdown）逐个调API生成test case JSON，支持追加模式和独立运行
- **测试矩阵生成**：本地生成Excel格式的测试矩阵（使用openpyxl，不再调用API）
- **版本编号**：自动检测output目录中已有的test_matrix_v*.xlsx文件，生成下一个版本号
- **命令行接口**：
  - `--input` / `-i`：指定输入文件路径（必需）
  - `--prompts` / `-p`：指定可选提示词组合，多个用逗号分隔（可选）
  - `--temp-prompt`：指定临时提示词内容（可选）
  - `-h`：打印帮助信息，动态列出所有可用的可选提示词
- **测试用例生成独立运行**：test_case_generator.py支持独立运行，可单独测试生成test case功能
  - `--analysis-dir`：指定分析结果目录（必需）
  - `--output`：指定输出的JSON文件路径（必需）
  - `--project-root`：指定项目根目录（可选，默认为脚本所在目录）
- **分析处理独立运行**：analysis_processor.py支持独立运行，可单独测试图片分析功能
  - `--images-dir`：指定图片目录（必需）
  - `--output-dir`：指定分析结果输出目录（必需）
  - `--prompts`：指定可选提示词组合，多个用逗号分隔（可选）
  - `--temp-prompt`：指定临时提示词内容（可选）
  - `--project-root`：指定项目根目录（可选，默认为脚本所在目录）
- **测试矩阵生成独立运行**：test_matrix_generator.py支持独立运行，可单独测试Excel生成功能
  - `--input-json`：指定test case JSON文件路径（必需）
  - `--output-dir`：指定输出目录（必需）
  - `--project-root`：指定项目根目录（可选，默认为脚本所在目录）

### 依赖库

- PyMuPDF (fitz)：PDF转图片
- openai：API调用
- Pillow：图片处理

### 改进

- **JSON解析容错增强**：test_case_generator.py的`_clean_json_response()`方法增强，处理LLM常见的JSON格式问题：
  - 移除Markdown代码块标记（```json...```）
  - 提取第一个`{`到最后一个`}`之间的内容（处理前后多余文本）
  - 移除JSON中的注释（`//`和`/* */`）
  - 处理单引号问题（将单引号替换为双引号）
  - 移除尾部逗号（在`}`或`]`前面的逗号）
  - 移除控制字符

### 已知限制

- 不支持Word/Excel/PPT文件格式（输出不支持信息）
- API模型名称固定为"gemma-4-local"，暂不支持配置
