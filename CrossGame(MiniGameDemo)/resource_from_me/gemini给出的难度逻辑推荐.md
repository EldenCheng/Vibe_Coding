# 🎮 小学六年级英语填字游戏：三级难度逻辑设计方案

本方案基于**“同一单词源（题库）”**原则，通过改变**线索暴露程度**、**逻辑推理要求**以及**干扰项机制**，专为小学六年级（11-12岁）的学生设计。旨在将死记硬背转化为“侦探式解谜”，兼顾小升初复习与游戏趣味性。

---

## 🟩 初级难度 (Primary)：基础记忆唤醒
**核心目标**：降低门槛，建立拼写信心，熟悉单词基础含义。

* **字母格状态**：所有单词的字母格完全空白，但明确显示字母数量（例如：`_ _ _ _ _`）。
* **中文提示**：严格按照游戏格子从上到下（或从左到右）的**先后顺序**给出最直接的中文翻译。
* **交互机制**：无限次尝试，无时间压力。
* **示例**：
    * 单词：`apple`
    * 格子：`[ ] [ ] [ ] [ ] [ ]`
    * 提示：1. 苹果

---

## 🟦 中级难度 (Intermediate)：侦探逻辑推理
**核心目标**：训练孩子利用已知线索推导未知线索的能力，告别按部就班。

* **限制 1：提示词乱序（核心逻辑）**
    * 下方的 4 个中文提示完全随机打乱。孩子需要根据格子长度或已填字母，自主匹配提示与格子。
* **限制 2：首尾字母“锚点”**
    * 为了平衡乱序的难度，每个单词的格子会自动固定显露**首尾两个字母**（如 `banana` 显示为 `b _ _ _ _ a`）。
* **限制 3：刚刚好的“字母备选池”**
    * 屏幕下方提供一个字母池，里面**仅包含**这 4 个单词当前所缺失的所有字母（打乱顺序）。
    * 当孩子正确填入一个字母后，该字母从备选池中消失。填到最后一个单词时，字母池基本清空，给予正向反馈。

---

## 🟥 高级难度 (Advanced)：小升初实战与抗压
**核心目标**：深化词义理解（结合句型与反义词），强化对易错拼写（如双写、元音）的肌肉记忆。

* **限制 1：多维语境提示（不直接给中文）**
    * **反义词/同义词法**：单词是 `heavy`（重的），提示给 `“light（轻的）的反义词”`。
    * **完形填空法（推荐）**：单词是 `weather`（天气），提示给 `“What's the ___ like today?”`。
* **限制 2：字母池“抓内鬼”（引入干扰项）**
    * 字母池不仅包含缺失字母，还会额外塞进 3-4 个**高频易错干扰字母**（例如：多给一个 `m` 或 `r` 来测试 `tomorrow` 的拼写）。
* **限制 3：盲盒格子（隐藏长度）**
    * 取消一格一格的方块限制，只给一条长下划线。孩子必须完全掌握单词长度，多写或少写均判定错误。
* **限制 4：限时“拆弹”模式**
    * 每关引入 60-90 秒倒计时，增加紧张感与挑战欲。

---

## 🛠️ 开发者通用数据结构 (JSON 示例)

你可以使用类似下面的单源数据结构，通过不同的字段直接驱动不同的难度关卡：

```json
{
  "word": "beautiful",
  "level_primary": {
    "hint": "美丽的，漂亮的",
    "display_type": "all_empty"
  },
  "level_intermediate": {
    "hint": "美丽的（注意：此提示将被 Shuffle 打乱）",
    "display_type": "show_fixed",
    "fixed_letters": {"0": "b", "8": "l"} 
  },
  "level_advanced": {
    "hint": "The opposite of 'ugly' (ugly的反义词)",
    "display_type": "underline_only",
    "distractor_letters": ["e", "a"]
  }
}
